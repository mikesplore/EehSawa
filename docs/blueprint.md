# **App Name**: EehSawa AI

## Core Features:

- Sarcastic Reply Generation: Generates a sarcastic reply based on user's input message, selected language (English, Kiswahili, or Sheng), and sarcasm level (Mild, Medium, Nuclear) using the Gemini API tool.
- Language Selection: Allows the user to select the language for the sarcastic reply: English, Kiswahili, or Sheng.
- Sarcasm Level Adjustment: Enables the user to adjust the level of sarcasm in the reply, choosing from Mild, Medium, and Nuclear options.
- Reply Output and Copy: Displays the generated sarcastic reply in a text box with a button to copy the text.

## Style Guidelines:

- Background: #0F172A (deep slate / near-black)
- Card / Surface: #111827
- Primary (CTA / highlights): #22C55E (confident green)
- Accent (sarcasm emphasis): #F59E0B (amber)
- Text (primary): #E5E7EB
- Text (muted): #9CA3AF
- Font: 'Space Grotesk', a sans-serif for a modern, slightly techy feel suitable for both headlines and content.
- Minimalist layout with a clean, straightforward design, focusing on ease of use and quick interaction.
- Use simple, recognizable icons for language selection and sarcasm level adjustment.
- Subtle, quick animations when the sarcastic reply is generated to give the app a lively feel.
